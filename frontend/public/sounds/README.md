# Sound Effects

Add the following sound files to this directory:

- `spin.mp3` - Looping wheel spin sound
- `tick.mp3` - Short tick when wheel stops
- `ding.mp3` - Success/win sound
- `boom.mp3` - Error/loss sound

Recommended sources:
- [Freesound.org](https://freesound.org)
- [Mixkit](https://mixkit.co/free-sound-effects/)
- [Pixabay Sound Effects](https://pixabay.com/sound-effects/)

Make sure sounds are short and punchy for the best mobile experience.
